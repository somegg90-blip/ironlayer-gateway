Quick Start

Clone the repo and run the container – your IronLayer proxy is live.

ironlayer-terminal
$ git clone https://github.com/somegg90-blip/ironlayer-gateway.git
$ cd ironlayer-gateway
$ docker build -t ironlayer .
$ docker run -p 8000:8000 -e API_KEY=your_key ironlayer
# proxy listening on http://localhost:8000
API_KEY=your_key – replace with your own secure key.
